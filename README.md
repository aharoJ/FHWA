## how to run

first, clone the repository to your local machine:

```sh
git clone https://github.com/aharoj/fhwa
```

```sh
cd fhwa
```

then, make the script executable:

```sh
chmod +x setup.sh
```

finally, run the script:

```sh
./setup.sh
```
